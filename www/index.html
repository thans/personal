Some recent work

Pinball Battle

While studying at KTH in Stockholm, Sweden, I took a computer game design course. We developed Pinball Battle using Unity, targeting the home console party game crowd. Pinball Battle is a multiplayer, free for all, pinball RTS game - gheck out the gameplay footage on the website!

  • About the game
  • Technologies: Unity, C#

Rally Software

As an intern at Rally Software for the summer of 2014, I was tasked with creating a fun data visualization. The Blast Down Chart represents data from Rally as a game (inspired by "Space Invaders").

  • Code on github
  • Technologies: Websockets, Angular JS, Ext JS, melon JS, html, css

Tor61

A partner project in CSE 461: Computer Networks. We created a simplified version of a TOR node to semi-anonymously browse the web.

  • Unfortunately, the code must be private on github
  • Technologies: Java (Networking!), TCP, Proxy, Lots of synchronization/threading

Fantasy Bachelor(ette)

I helped create a game like "fantasy football" but for ABC's "The Bachelor" and "The Bachelorette" with one of my friends from UW. I mostly worked on the database side. The site used to be live at www.fantasybach.com for the last 2 seasons of the show

  • Technologies: Node js, javascript, MySQL
